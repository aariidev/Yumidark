## 📌 Descripción

Yumidark es un potente bot de Discord diseñado para **profesionales y entusiastas de la ciberseguridad**. Permite realizar pruebas de penetración éticas, escaneos de vulnerabilidades y análisis de seguridad directamente desde Discord de forma rápida y controlada.

Ideal para pentesters, red teams, bug bounty hunters y administradores de sistemas que necesitan herramientas de seguridad al alcance de un comando.

---

## ✨ Características Principales

- 🔍 **Escaneos avanzados** de puertos, servicios y vulnerabilidades
- 🛡️ **Análisis de seguridad** en tiempo real
- 📊 **Reportes detallados** de hallazgos
- ⚡ **Comandos rápidos** y fáciles de usar
- 🔒 **Enfoque en seguridad ética** y responsible disclosure
- 📱 **Totalmente integrado** en Discord (slash commands)

---

## 🚀 Cómo Invitar a Yumidark

1. Haz clic en el botón de arriba o visita:  
   [Invitar Yumidark](https://discord.com/discovery/applications/1514847704075862056)
2. Selecciona el servidor donde quieres añadirlo
3. Otorga los permisos necesarios (recomendado: solo a roles de confianza)

> **Importante**: Solo invita el bot a servidores donde tengas permiso para realizar pruebas de seguridad.

---

## 🧰 Comandos Principales

Usa `/help` dentro de Discord para ver la lista completa y actualizada de comandos.

Ejemplos comunes:
- `/scan` - Iniciar un escaneo de seguridad
- `/nmap` - Ejecutar escaneo de puertos (con opciones)
- `/vuln` - Buscar vulnerabilidades conocidas
- `/report` - Generar reporte del último escaneo
