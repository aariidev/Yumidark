# Política de Privacidad de Yumidark

**Proyecto de Ciberseguridad**

* Última actualización: 12 de junio de 2026*

## 1. Introducción

En Yumidark, nos tomamos muy en serio la privacidad y la seguridad de los datos. Esta Política de Privacidad explica cómo recopilamos, usamos y protegemos tu información cuando interactúas con el proyecto.

## 2. Información que Recopilamos

Como proyecto open source alojado en GitHub:
- **Datos de GitHub**: Cuando contribuyes (issues, pull requests, comentarios), GitHub recopila tu nombre de usuario, email y actividad. Consulta la [Política de Privacidad de GitHub](https://docs.github.com/en/site-policy/privacy-policies/github-privacy-statement).
- **No recopilamos datos personales directamente**: A menos que implementes servicios adicionales (como un bot de Discord o API), Yumidark no almacena datos de usuarios finales.
- Si en el futuro se añade funcionalidad que recopile datos (ej. logs de seguridad, reportes), se notificará aquí.

## 3. Cómo Usamos la Información

- Para mantener y mejorar el proyecto.
- Para comunicarnos contigo sobre contribuciones o issues de seguridad.
- Para análisis de seguridad y mejora de prácticas (datos anonimizados).

## 4. Compartir Información

- No vendemos ni compartimos datos personales con terceros.
- Los datos de contribuciones son públicos en GitHub.
- En caso de requerimiento legal, cooperaremos con autoridades competentes.

## 5. Seguridad de los Datos

Implementamos y promovemos las mejores prácticas de ciberseguridad:
- Código revisado para seguridad.
- Uso de herramientas de análisis estático y dinámico.
- Recomendamos a los usuarios seguir principios de zero-trust y encriptación.
- Cualquier incidente de seguridad se manejará con disclosure responsable.

## 6. Cookies y Tecnologías de Seguimiento

Este repositorio no utiliza cookies. Si se desarrolla un sitio web o dashboard asociado, se actualizará esta sección.

## 7. Tus Derechos

Según el RGPD (UE):
- Derecho de acceso, rectificación, eliminación, etc.
- Contacta para ejercerlos.

## 8. Cambios en esta Política

Actualizaremos esta política según evolucione el proyecto. Revisa periódicamente.
